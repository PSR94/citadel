# Privacy Data Handling Standard

Teams handling personal data must classify the data, minimize unnecessary replication, and document approved transfer paths.

## Required controls

- use approved systems for regulated data workflows
- avoid copying personal data into ad hoc collaboration tools
- escalate suspected exposure through the Security Incident Response Playbook
- retain only the minimum records required by the Data Retention Standard or an approved exception
