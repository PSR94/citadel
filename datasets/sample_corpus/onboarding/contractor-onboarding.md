# Contractor Onboarding Manual

Contractors receive time-bound access tied to a sponsoring manager, a scoped statement of work, and stricter sharing boundaries for personal data.

## Required steps

1. confirm a sponsoring manager and contract end date
2. complete identity proofing and confidentiality terms
3. enroll either a managed device or approved virtual workspace
4. receive least-privilege access through Access Broker
5. review handling restrictions for customer and employee data

## Access profile

- access expires automatically on the recorded end date
- production access is not granted by default
- directory visibility is limited to the contractor's assigned org units
- hardware issuance may be replaced by a managed virtual workspace

## Differences from employee onboarding

Contractors operate under time-boxed access, narrower directory visibility, and stronger sponsor review. Employees receive payroll enrollment and standard managed workstation issuance as the default path.
